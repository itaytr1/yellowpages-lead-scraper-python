# 🟡 Python Yellowpages Lead Scraper (API)

Are you tired of getting IP-blocked, dealing with Cloudflare CAPTCHAs, and managing expensive residential proxy rotations just to extract local B2B leads?

This repository demonstrates how to easily extract highly structured, real-time local business data (Names, Phones, Websites, Review Counts, and Categories) using the **Premium Local Business Leads API**.

### 🚀 Get Started Immediately
Stop writing complex BeautifulSoup loop scripts that break every time Yellowpages updates their HTML. Get your API key and start pulling clean JSON data in 60 seconds:
👉 **[https://rapidapi.com/itaytr1/api/premium-local-business-leads]**

### Why use this API instead of scraping it yourself?
* **Anti-Bot Bypass:** We handle the residential proxy rotation and TLS fingerprinting. You never get a 403 Forbidden error.
* **Auto-Pagination:** Pull up to 100 leads per request without writing loop logic.
* **Deep Enrichment:** We automatically extract star ratings, years in business, and specific service categories.
* **Clean JSON:** Ready to inject directly into Salesforce, HubSpot, or your cold-email software.
